Select2 Attribute Plugin

Zero-JS Select2 configuration via HTML data attributes

1 — Static Options
No JS needed — just data-select2-attr on the element.
Selected: —
data-select2-attr
data-placeholder="Select a country..."
data-allow-clear="true"
2 — AJAX Data Source
Fetches from /api/users?search= — type to filter.
Selected: —
data-url="/api/users"
data-data-path="data"
data-id-key="id"  data-text-key="name"
data-on-select="callbacks.onUserSelect"
3 — Pagination
Scroll down in the dropdown to load more pages.
data-pagination="true"
data-page-size="5"  data-total-path="data.total"
data-page-param="page"  data-page-size-param="limit"
4 — Dependency (parent → child)
Disabled until a country is selected. Clears on parent change.
data-depends-on="#s-country"
data-dep-param="countryId"  ← sent as ?countryId=
5 — Initial Value (data-current-id)
On load, fetches /api/products?id=3 and pre-selects the result.
data-current-id="3"
data-fetch-url="/api/products"
data-fetch-id-param="id"
6 — on-change & watch
Clears when the user select changes.
← trigger element:  data-on-change="#s-watched-product"
← watched element: data-watch="#s-trigger"  data-watch-action="clear"
7 — Multiple Selection & Tags
multiple select:  multiple attribute on <select>
tags mode:        data-tags="true"
8 — Auto-Init in Modal (dynamic elements)
Select2 inside the modal auto-inits and uses the modal as dropdownParent.
Modal — Auto Init
dropdownParent is automatically set to .modal.
